# Simple Calculator

A basic web-based calculator application built using HTML, CSS, and vanilla JavaScript. This project demonstrates fundamental web development concepts and client-side logic for performing arithmetic operations.

## Features

-   Addition, Subtraction, Multiplication, Division.
-   Clear function to reset the calculator.
-   Decimal point support.
-   Responsive design (basic).

## Project Structure

```
.
├── index.html
├── style.css
├── script.js
├── package.json
└── README.md
```

## Installation

This project is a static web application and does not require any backend dependencies. You only need a web browser to run it.

1.  **Clone the repository (if applicable):**
    ```bash
    git clone https://github.com/your-username/simple-calculator.git
    cd simple-calculator
    ```
2.  **Install Node.js (Optional but recommended for `npm` scripts):**
    If you don't have Node.js installed, you can download it from [nodejs.org](https://nodejs.org/). This will allow you to use `npm` commands.

## Setup Commands

After cloning the repository and ensuring you have Node.js installed:

```bash
npm install
```
This command will initialize a `package-lock.json` file. For this project, it's mostly a formality as there are no external dependencies.

## How to Run the Project

You can run the calculator in two ways:

### 1. Directly open `index.html`

Simply navigate to the project directory and open the `index.html` file in your preferred web browser.

### 2. Using `npm` script

If you have Node.js installed, you can use the provided `npm` script to open the calculator in your default web browser:

```bash
npm start
```
This command attempts to open `index.html` using the appropriate command for your operating system (macOS, Linux, or Windows).
